# 个人收集Library

[google/guava，里面的 IO、集合、并发、事件总线 都值得一用](https://github.com/google/guava)

[google/truth，断言类工具，比Assert丰富许多](https://github.com/google/truth)

[google/error-prone，一款maven插件，将运行时错误转换成编译时错误，以便提早发现问题](https://github.com/google/error-prone)

[looly/hutool，一款不错的工具类合集](https://github.com/looly/hutool)

[binarywang / java-emoji-converter，一款emoji表情转换工具，以便存储到数据库](https://github.com/binarywang/java-emoji-converter)

[Wechat-Group/WxJava，by binarywang，微信开发必看](https://github.com/Wechat-Group/WxJava)

[2019年度 最受欢迎中国开源软件](https://www.oschina.net/project/top_cn_2019?utm_source=startnews)

[alibaba/transmittable-thread-local，线程切换时支持传递threadLocal持有值给子线程](https://github.com/alibaba/transmittable-thread-local)

[alibaba/easyexcel，超好用的excel解析、生成工具](https://github.com/alibaba/easyexcel)

[alibaba/jetcache，据说提供了比 Spring Cache 更加丰富的注解支持功能](https://github.com/alibaba/jetcache)

[alibaba/canal，阿里巴巴 MySQL binlog 增量订阅&消费组件](https://github.com/alibaba/canal)

[alibaba/otter，阿里巴巴分布式数据库同步系统(解决中美异地机房)](https://github.com/alibaba/otter)

[alibaba/yugong，阿里巴巴从Oracle到Mysql数据迁移同步工具(全量+增量,目标支持MySQL/DRDS)](https://github.com/alibaba/yugong)

[SpringForAll/spring-boot-starter-swagger，swagger的封装starter，方便使用](https://github.com/SpringForAll/spring-boot-starter-swagger)

[vjtools，唯品会 的一款工具类库，内含线上诊断的工具类](https://github.com/vipshop/vjtools)
